# Luma – Emotional Support Chatbot

An AI-powered companion that offers empathetic conversations, coping strategies, and mental wellness resources—built with Next.js, Tailwind CSS, and framer-motion.

## ? Features
- Initial emotion check-in (""How are you feeling?"")
- Adaptive responses based on mood
- Mindfulness & journaling prompts
- Privacy-first design (no data stored)
- Mobile-responsive UI

## ?? Based on PRD
Aligned with clinical best practices for non-diagnostic emotional support (see [chatbot_PRD.pdf](chatbot_PRD.pdf)).

## ?? Built With
- Next.js 16 (App Router)
- Tailwind CSS
- Framer Motion
- React Icons
